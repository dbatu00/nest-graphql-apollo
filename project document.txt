This is a simple CRUD website

It has 3 functionalities:
    -get users
    -add user
    -delete users

    get: 
    -can be run with: 
        -username strings separated by commas
        -user ids separated by commas
        -no input to get all users

    *if there is an input it must be all strings or all ids

    add:
    -input username to add it to DB
    -




To run the project: on root:  $npm run dev
